Verbindung zum Router herstellen

Klicken Sie weiter unten auf 'Bestätigen', um Ihren Router Rebooter mit dem Wi-Fi Ihres Routers zu verbinden. Es kann einige Minuten dauern, bis der Router Rebooter eine Verbindung aufgebaut hat.

Nutzen Sie die LEDs auf Ihrem Router Rebooter, um den Fortschritt zu verfolgen.

  • Blinkendes rotes LED-Licht:
    Wartet darauf, Verbindung zu Ihrem Router aufzubauen
  • Blinkendes blaues LED-Licht:
    Verbindung zu Ihrem Router wurde hergestellt, das Internet wird NICHT überwacht
  • Permanent leuchtendes blaues LED-Licht:
    Verbindung zu Ihrem Router wurde hergestellt, das Internet WIRD überwacht

Sobald die Verbindung aufgebaut wurde, sollte Sie die Übersichtsseite des Rebooters innerhalb von 20 Minuten über Ihr Mobilgerät aufrufen:

  • iOS: Verbinden Sie Ihr Mobilgerät mit Ihrem Router. Rufen Sie über Safari http://zobox.local auf.
  • iOS/Android: Verbinden Sie Ihr Mobilgerät erneut mit dem Wi-Fi (Zobox-ABCD) Ihres Rebooters und rufen Sie dann über Safari/Chrome http://192.168.4.1 auf.

Besuchen Sie unsere Webseite http://www.zobox.xyz/help falls Sie Hilfe benötigen.
